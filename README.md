# PVPBlaze

A simple websaite made for Minecraft server made with NextUI v1.

## Live Demo
[Click here](https://pvpblaze.net)

## Getting Started

```bash
git clone https://github.com/cmclient/pvpblaze-website
cd pvpblaze-website
npm install
npm run dev
```
